# Stone Grey Website

This is a minimal static website for `http://stonegrey.com/`.

## Files

- `index.html` is the home page.
- `styles.css` contains all styling.
- `CNAME` tells GitHub Pages to serve the site at `stonegrey.com`.

## Deploy To GitHub Pages

This folder can be published as a GitHub Pages site from the root of the `main` branch.

Suggested repo: `duncan-clark/stonegrey.com`.

After the repo is pushed, configure Pages to deploy from `main` / root and point the domain DNS to GitHub Pages.
